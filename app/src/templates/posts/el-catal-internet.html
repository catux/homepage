{% extends 'post_template.html' %} {% block title %} El català a Internet {% endblock %} {% block author %} cpina {% endblock %} {% block post_date %} 2005-10-23 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Es pot llegir a un estudi de Jordi Mas, per a Softcatalà, l'evolució del català a Internet en els darrers anys, del 2003 al 2005, concretament.

L'estudi, totalment empíric, cal posar-lo subjecte a moltes variants (s'utilitza el motor de búsqueda de Google, que pot haver millorat amb el temps, indexant més pàgines d'un llenguatge que d'altres per qualsevol motiu, etc.)

Es veu que tenim aproximádament una pàgina per catalano-parlant, lluny de l'anglès (el motiu segur que és el gran nombre de pàgines angleses, fetes fins i tot per gent que no té l'anglès com a llengua mare).

Paral·lelament, s'ha fet l'ordenació de quins països han tingut més creixement i pel creixement ponderat pel número d'habitants. Com a curiositat he marcat les llengues dels països que els últims anys s'han independitzat (si més no, Estonià, Eslové i Lituà). Personalment m'ha sorprés que el creixement ponderat a número d'habitants és dins dels 6 més alts. {% endblock %} {% block comments %}
