{% extends 'post_template.html' %} {% block title %} L4: Els microkernels del present {% endblock %} {% block author %} jvprat {% endblock %} {% block post_date %} 2004-10-11 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Avui estava tafanejant sobre GNU Hurd, un nucli (basat en el microkernel GNU Mach) desenvolupat per GNU per suplir Linux i aconseguir un sistema operatiu totalment GNU, i he anat a parar de rebot a la pàgina de L4.

L4 és una família de microkernels (Fiasco, L4Ka::Pistachio, etc...) desenvolupats en paral·lel per diverses universitats, que comparteixen la mateixa API.

El principal avantatge sobre GNU Mach és que aquests mantenen un desenvolupament força actiu, i per tant aconsegueixen uns resultats molt més visibles (a curt termini). L'exemple per excel·lència és L4Linux, una adaptació de Linux que utilitza els microkernels L4, de la que fa poc han tret la versió 2.6, basada en Linux 2.6.8.1. Per als més agosarats, hi ha fins i tot un CD de demostració. Si algú el prova ja comentarà com ha anat l'experiència ;)
