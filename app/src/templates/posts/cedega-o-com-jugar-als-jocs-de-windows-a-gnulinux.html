{% extends 'post_template.html' %} {% block title %} Cedega: O com jugar als jocs de Windows a GNU/Linux {% endblock %} {% block author %} Pinucset {% endblock %} {% block post_date %} 2006-05-26 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Cedega és una aplicació que permet jugar a jocs només suportats a l'entorn Windows a GNU/Linux d'una manera molt fàcil i la majoria de vegades transparent.

Té una interfície anomenada Point2Play en la qual tot és molt fàcil, s'instal·len els jocs seleccionant el Setup.exe del CD del joc i s'executa fent clic a una icona que ell crea.

TransGaming (empresa que s'encarrega de Cedega) permet que els usuaris votin per escollir quin joc volen que sigui suportat i també ofereix servei tècnic.

Malauradament és una aplicació privativa i que s'ha de pagar una quota mensual de 5 euros, compilar de CVS (difícil) o baixar-se la demo (15 dies) per tal de poder-la usar.

Completament recomenable per jugar des de GNU/Linux. {% endblock %} {% block comments %}
