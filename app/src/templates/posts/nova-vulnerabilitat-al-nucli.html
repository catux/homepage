{% extends 'post_template.html' %} {% block title %} Nova vulnerabilitat al nucli. {% endblock %} {% block author %} seoman {% endblock %} {% block post_date %} 2004-09-19 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} S'han detectat dues vulnerabilitats al nucli de Linux. Aquestes dues vulnerabilitats, tan poden servir per bloquejar la màquina atacada com per permetre executar codi arbitrari com a usuari root.

Les dues vulnerabilitats són:

* Un desbordament en el codi del servidor nfs. Aquest desbordament tant pot permetre el bloqueig com l'execució de codi arbitrari.

* Un usuari local pot penjar la màquina vctima manipulant els fitxers especials "/dev/ptmx".


Encara no s'han publicat els pegats per solucionar això, però les versions "2.6.9-rc2" o "2.4.28-pre3" ja ho tenen corregit, tot i que són beta.
