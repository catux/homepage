{% extends 'post_template.html' %} {% block title %} Com canviar els botons de les finestres a GNOME {% endblock %} {% block author %} Pachu {% endblock %} {% block post_date %} 2005-05-25 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} A vegades, ens pot interesar canviar l'ordre dels botons de minimitzar, maximitzar, tancar i opcions a les finestres.
Doncs en GNOME és molt senzill de fer.

Hem d'anar al menú del quadre:

Aplicacions->Sistema->Editor de la configuració

Un cop allà, anirem a apps->metacity i veurem que si despleguem metacity i seleccionem general, apareix l'opció button_layout.

El valor de button_layout és:

menu:minimize,maximize,close

Doncs només haurem de canviar-los, on els dos punts separa la dreta de l'esquerra.

Així serà fàcil emular les finestres d'un MacOSX. {% endblock %} {% block comments %}
