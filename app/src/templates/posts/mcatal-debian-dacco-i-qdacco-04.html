{% extends 'post_template.html' %} {% block title %} Més català a Debian: Dacco i qdacco 0.4 {% endblock %} {% block author %} cpina {% endblock %} {% block post_date %} 2006-04-20 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Fa molt temps es va comentar aquí mateix de l'existència del projecte Dacco, i també d'un programa anomenat qdacco.

Recordar que Dacco és un diccionari anglès-català i català-anglès amb llicència LGPLG, que s'ha construït des de zero amb l'aportació de gent. Ara per ara té més de 10.000 entrades amb cada llengua, exemples, notes d'ús, etc. qdacco és una interfície gràfica per accedir en local a les dades del diccionari, apart d'enviar suggeriments, paraules noves, etc.

Ja fa unes setmanes que Dacco està disponible dins Debian unstable i testing, junt amb qdacco.

A més a més fa pocs dies s'ha alliberat la versió 0.4 de qdacco, que afegeix funcionalitats força interessants respecte les versions anteriors i una interfície gràfica molt millorada.

La nova versió es pot trobar a la Web de qdacco com dins mateix Debian.
Així per instal·lar-la només cal un apt-get install qdacco (o aptitude install qdacco). A la seva Web es pot descarregar el codi font per compilar-ho per altres distribucions.
