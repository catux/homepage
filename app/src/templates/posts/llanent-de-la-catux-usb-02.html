{% extends 'post_template.html' %} {% block title %} Llançament de la Catux-USB 0.2 {% endblock %} {% block author %} jvprat {% endblock %} {% block post_date %} 2006-02-02 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Darrerament hem canviat el plantejament de les versions de la Catux-USB. A partir d'ara intentarem centrar-nos en objectius a més curt termini i treure versions amb més freqüència, que en cap cas estan pensades com a producte final, sino més aviat com a versions de prova.

Seguint aquesta idea hem tret la versió 0.2 de la Catux-USB, que és una reorganització del projecte, intentant fer-lo més mantenible i ampliable.

Per als agosarats que volgueu provar aquesta nova versió, la podeu baixar d'aquí: ftp://ftp.caliu.info/pub/distribucions/catux/

Les característiques d'aquesta versió es poden consultar al Trac: http://catux.org/cgi-bin/trac.cgi/milestone/0.2

Recordo una vegada més que es tracta tan sols d'una versió de desenvolupament. Si esteu interessats en col·laborar, baixeu-vos-la, i envieu els vostres comentaris, suggerències, etc. a la llista de distribució de la Catux-USB. {% endblock %} {% block comments %}
