{% extends 'post_template.html' %} {% block title %} Projecte Orange {% endblock %} {% block author %} cpina {% endblock %} {% block post_date %} 2005-08-17 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Blender és un conegut programa de disseny 3D, el qual fa anys que és lliure.

Últimament s'ha potenciat molt les seves capacitats i encara ho farà més. El nivell és equivalent al de 3DStudio.

Fa uns 15 dies es va posar en marxa el projecte Orange, el qual té per objectiu fer una pel·lícula per part de les 6 persones més involucrades en Blender, però per això necessiten una quantia de diners on tothom hi pot col·laborar comprant el DVD que conté el resultat final (pel·lícula a alta definició), el procés de creació i tots els fitxers font del disseny.

Si es col·labora, amb un import de 35€ abans de Setembre s'inclou el vostre nom als crèdits de la pel·lícula.

Fer la pel·lícula també potenciarà l'apartat d'animació de Blender, que actualment és dels més fluixos i quedarà al nivell suficient per fer una bona pel·lícula.

El pla de treball és que es "tancaran" les 6 persones del projecte a Amsterdam a treballar-hi durant uns 6 mesos.

Actualment porten 414 compres del DVD (donacions) i s'hauria d'arribar a 1000 donacions.

Es pot veure el gràfic de les donacions a la pàgina principal de Blender. {% endblock %} {% block comments %}
