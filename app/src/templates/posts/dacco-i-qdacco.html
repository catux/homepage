{% extends 'post_template.html' %} {% block title %} Dacco i QDacco {% endblock %} {% block author %} cpina {% endblock %} {% block post_date %} 2005-09-25 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Fa uns dies, via el blog de Xavi Caballé (i notícia a Puntbarra) vaig descobrir el projecte Dacco, el qual és un projecte que permet tenir un diccionari català-anglès i anglès-català de forma totalment lliure (feta per la comunitat, sense basar-se en altres diccionaris, etc.)

Vaig trobar a faltar una eina per consultar les dades sense connexió (es podia fer amb PDF i amb Firefox). Així que vàrem començar a fer el QDacco, el qual és una interfície amb les Qt's (les llibreries gràfiques que utiltiza KDE), totalment funcional, tant per Linux com per Windows.

Aquí en podeu llegir la referència amb instruccions. Si teniu qualsevol problema, sugeriment, etc. us podeu posar en contacte amb mi (correu electrònic o deixeu un comentari aquí)

És, tal com es comenta a la notícia de Dacco, la versió 0.2. Hauria de ser funcional sense problema, però no ha estat (encara) ampliament provada.
