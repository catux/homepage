{% extends 'post_template.html' %} {% block title %} Diumenge 17, trobada a la UPC de Manresa {% endblock %} {% block author %} jvprat {% endblock %} {% block post_date %} 2005-07-16 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Avui dissabte 16, s'ha realitzat la primera "jornada wireless" de la comunitat Guifi Bages, en la que Catux està implicada.

Aquest dilluns es fa el muntatge del primer supernode, i encara no hem polit el firmware que han de portar els routers. És per això que els 4 representants de Catux que erem a la trobada hem decidit que demà a la tarda aniriem a fer feina per avançar en aquesta tasca, per tenir-ho tot a punt per dilluns.

Per això hem decidit canviar el lloc de la trobada presencial d'aquest diumenge a la UPC de Manresa, i hi serem a partir de les 3. Segurament no es tractaran els temes habituals (ni podrem estar gaire per solucionar problemes), però serà una experiència enriquidora.
