Ara acabo d'arrancar l'xfreedesktop, que és una lliure implementació de l'xfree86 4.4.0 i, degut això, no perdem qualitat envers l'xfree86... De fet, aparentment, no hi ha diferència :-P
Per Debian, hi ha un repositori 'no oficial':

deb http://freedesktop.org/~jamey/debian testing main

Aviso que si teniu l'xfree86 instal·lat (segurament), feu:

# apt-get install -d fd.o-* -y

i després anar a /var/cache/apt/archives/ i fer:

# dpkg --force- -i "paquet"

amb tots els que us heu baixat abans perquè sobreescriuran algunes coses de l'xfree ;-D

Sino sempre podeu baixar les fonts de http://x.org/ i a http://www.badopi.org/node/view/523 teniu com fer-ho {% endblock %} {% block comments %}
