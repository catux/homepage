{% extends 'post_template.html' %} {% block title %} CaTuX-USB {% endblock %} {% block author %} Pachu {% endblock %} {% block post_date %} 2004-07-09 00:00:00 {% endblock %} {% block categories %} Notícies, Últimes notícies {% endblock %} {% block post %} Coincidint amb la ponència que realitzarem dissabte a les 10 del matí a les jornades de programari lliure, des de Catux, anunciem el llançament de la distribució que s'ha creat a la nostra associació.

El seu nom és Catux-USB, una distribució pensada per a dispositius USB (amb arrancada des d'el mateix USB) basada, en un principi, en Knoppix (basada al mateix temps en Debian), però modificada de tal manera que ens ofereix les prestacions d'una distribució completa, amb l'eina apt-get totalment operativa amb la peculiaritat de poder escriure en tot el sistema. A part d'altres propietats que ja veureu ;)

En resum, un sistema portàtil i complet.

Ja us comunicarem d'on la podreu descarregar.

Esperem que la testejeu i envieu els vostres comentaris i els possibles bugs que pogueu trobar. :) {% endblock %} {% block comments %}
